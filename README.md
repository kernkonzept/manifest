# L4Re manifest

This package contains the ham manifest which can be used to checkout and manage
the L4Re operating system git repositories.

## The L4Re Operating System

The [L4Re Operating System Framework](https://l4re.org/) provides the
building blocks for systems with real-time, security, safety and
virtualization requirements. L4Re is a microkernel architecture and provides the
infrastructure and components needed to implement any such architectures.

Please find more information on [l4re.org](https://l4re.org).
